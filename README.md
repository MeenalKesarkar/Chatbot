# 🤖 Chatbot Web App

A simple and interactive **Chatbot Web Application** built using **React.js**.
This app allows users to send messages and receive automated responses, including dynamic features like current date and time.

---

## 🚀 Features

* 💬 Interactive chat interface
* 🤖 Automated bot responses
* 📅 Displays current date dynamically
* ⏰ Shows current time
* ⚡ Fast and responsive UI
* 🎯 Beginner-friendly project structure

---

## 🛠️ Tech Stack

* ⚛️ React.js
* 📜 JavaScript (ES6+)
* 🎨 CSS3
* 🔗 Axios (if used in future upgrades)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/chatbot-app.git
```

2. Navigate to project folder:

```
cd chatbot-app
```

3. Install dependencies:

```
npm install
```

4. Run the app:

```
npm run dev
```

---

## 💡 How It Works

* User enters a message
* Message is stored in state
* Bot processes input using custom logic
* Response is displayed instantly
